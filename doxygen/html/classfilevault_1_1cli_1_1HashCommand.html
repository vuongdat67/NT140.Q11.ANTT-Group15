Detailed Description

Hash command - Calculate cryptographic hashes.

Supports: MD5, SHA1, SHA2 family (224/256/384/512), SHA3 family (224/256/384/512), BLAKE2 (b/s variants)

Features:

  • HMAC mode with key
  • Hash verification
  • Batch processing
  • Performance benchmarking
